What is a DDoS Attack?

A Distributed Denial of Service also known as a DDoS attack is an attack that attempts to disrupt the normal traffic of a targeted server, service, or network by overwhelming it with a flood of visits often through compromised networks.

Unlike a DoS attack, which is usually from a single source, DDoS attacks use lots of compromised devices to generate the fake traffic, making it harder to mitigate.

Types of DDoS Attacks

Some common types of DDoS attacks include:

  • Volumetric Attacks: Volumetric attacks like the name suggests floods the target with a high volume of traffic to consume its bandwidth. Some examples include UDP and ICMP floods.
  • Protocol Attacks: These attacks use network protocols to exhaust server resources. Some examples include SYN floods and Ping of Death.
  • Application Layer Attacks: These attacks target the application layer of the network for example by using HTTP floods that overwhelm the server.

Mitigation Techniques

Common ways to mitigate DDoS attacks include:

  • Using some DDoS protection services that detect and filter out malicious traffic (if too strict this could block normal traffic so be careful).
  • Implementing rate limiting to reduce impact of incoming requests.
  • Monitoring network traffic to quickly identify and respond to unusual spikes.
